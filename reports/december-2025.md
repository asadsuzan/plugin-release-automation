# Monthly Report — December 2025

## Section Separator (v2.0.0)

### Features
- **title: Added four new aesthetic wave presets: Ocean, NeonGlow, Fire, and Sunset.**
- **Added support for controlling wave amplitude (height/intensity).**
- **title: Added Wave Direction options: Left-to-Right (ltr), Right-to-Left (rtl), Invert, Static, Bounce.**
- **Added vertical flip option to invert wave direction vertically.**
- **title: Added a brand-new Separator Type: Pill, featuring colorful floating pill labels.**
- **Pill separator includes smooth infinite marquee animation with customizable duration/speed.**
- **title: Each pill supports individual customization: Background color, Text color, Label text, Rotation angle.**
- **title: Global style controls for the pill track: Padding, Margin, Background, spacing/gap.**
- **title: Individual pill item styling controls: Padding, Shadow, Typography, Border, Border Radius.**
- **Added a brand-new BoldGradient Separator combining editable pill-style items with pattern and wave overlays.**
- **title: Background Type control: Both, Pattern, Organic, or None.**
- **title: Organic Waves Controls: Enable/disable, Top & Bottom Opacity, Wave Height, Color Mode (Solid/Gradient), Gradient Options (Start Color, End Color, Direction: Horizontal, Vertical, Diagonal).**
- **title: Pattern Controls: Toggle visibility, Opacity, Configure Angle, Transparent Gap, Stripe Colors, and Distances.**
- **title: Dynamic Gradient Items: Inline RichText editing, Add/Duplicate/Remove items, Per-item color control.**
- **title: Gradient Item Styling Controls: Padding, Border, Border Radius, Shadow.**
